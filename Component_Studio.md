Component Studio — Dashboard Features (High-Level)
 1. Main Dashboard (Entry Screen)
Clean landing screen after login
Shows different ways to create components
Minimal, modern UI (card-based options)
Each option represented as a selectable card
2. Creation Modes (Core of Dashboard)
        Manual Builder (MVP — MUST HAVE)
        “Start from scratch”
        Opens drag & drop editor
        User builds component manually
3. Guided Builder (Question-Based UI Generation)
User selects “Describe what you want”
System asks structured questions like:
What type of component? (card, form, etc.)
What fields do you need?
Layout preference?
Colors/style?
Based on answers → generate initial UI
Then open in editor for customization

This is Phase 2 (very strong differentiator)

AI Builder (Future — Not required now)
Upload image OR describe UI
Answer a few questions
System generates UI automatically

This is Phase 3 (advanced feature)
